Extensions


The functionality of Tuft can be extended via prehandlers and responders. Below is a list of current first-party Tuft extensions, in addition to those that are already built in to the core library.

Prehandlers

  • Helmet - A wrapper for the popular Express security middleware package.

Responders

  • View - Enables the use of template engines for rendering views.
