Tuft

An HTTP/2 web framework for Node.js

$ npm install tuft

Speedy

Tuft is an HTTP/2-only framework built around Node's core HTTP/2 API, which gives it a performance advantage over other frameworks that rely on Node's compatibility API.


See the benchmarks.

Minimalist

Routing is baked into the core functionality, but many other features commonly found in other route-based frameworks are intentionally not included, which results in minimal overhead.


Scalable

Tuft's routing algorithm has a timing complexity of O(1), so it will continue to serve requests at lightning fast speed, even with thousands of routes.

Extensible

Add the required functionality to your server only as needed. Utilize Tuft's built-in extensions or write your own, and optionally make them available to other developers.
