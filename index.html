Tuft

A fast, lightweight web framework for Node.js

$ npm install tuft

Speedy

Tuft is one of the fastest web frameworks for Node.js. See the benchmarks.

Minimalist

Routing is a core feature of Tuft, but features commonly found in other route-based frameworks are intentionally not included, resulting in minimal overhead.


Scalable

Tuft's routing algorithm has a timing complexity of O(1), so it will continue to serve requests at lightning fast speed, even with thousands of routes.

Extensible

Add the required functionality to your server only as needed. Utilize Tuft's built-in extensions or write your own, and optionally make them available to other developers.
