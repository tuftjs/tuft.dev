Tuft

An HTTP/2 web framework for Node.js

$ npm install tuft

Speedy

Tuft is an HTTP/2-only framework built around Node's core HTTP/2 API, which gives it a performance advantage over other frameworks that rely on Node's compatibility API.


See the benchmarks.

Minimalist

A basic Tuft server is dependency-free, and only comes with a small set of features to keep things lean. Routing is baked into the core functionalty, but most other features that are commonly found in other route-based frameworks are intentionally not included, which results in minimal overhead.


Scalable

Tuft's routing algorithm has a timing complexity of O(1), so it will continue to serve requests at lightning fast speed no matter how many routes you add.

Extensible

Add the required functionalty to your server only as needed. Features like request body parsing and response body streaming are packaged as separate extensions within Tuft's base installation. You can also create your own extensions, and optionally make them available to other developers.
