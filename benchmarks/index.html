Benchmarks


This page contains the benchmark results for Tuft compared with other popular Node.js web frameworks. For more information on how the benchmarks were set up, and to view the specific application code used for each framework, see the GitHub repository.

The benchmark results below were obtained using AutoCannon, and are ordered from fastest to slowest.

Results

  • Date: September 16, 2020
  • Hardware: Digital Ocean CPU-optimized droplet
  • OS: Ubuntu 20.04 (LTS) x64
  • Node version: 12.18.4
  • Client command: autocannon -c 100 -d 40

Framework Version Req/s Bytes/s (MB) Latency (ms)
none N/A 22,474 3.48 4.08
Tuft 2.3.1 20,872 3.23 4.17
Fastify 3.4.1 18,858 2.92 5.08
Koa 2.13.0 14,959 2.32 6.14
