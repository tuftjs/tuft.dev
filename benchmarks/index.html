This page contains the benchmark results for Tuft compared with a few other popular web frameworks that also support HTTP/2. For more information on how the benchmarks were set up, and to view the specific application code used for each framework, see the GitHub repository.

The benchmark results below use an HTTP/2 load testing client called h2load, which comes bundled with nghttp2.

  • Date: May 14, 2020
  • Hardware: Digital Ocean CPU-optimized droplet (4 vCPUS, 8GB)
  • OS: Ubuntu 18.04.3 (LTS) x64
  • Node version: 12.16.3
  • Client command: h2load -c100 -m100 --duration=60 --warm-up-time=30
